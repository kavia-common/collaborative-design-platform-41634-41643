# Figma-like Editor (React)

This frontend implements the initial scaffold for a Figma-style editor:
- Left sidebar navigation
- Top action bar
- Central SVG canvas with basic tools (select, rectangle, ellipse, text)
- Right inspector with Layers and Properties

Routes:
- `/` Dashboard
- `/editor` Editor
- `/components` Components library placeholder

Quick start:
- `npm install`
- `npm start` (runs on port 3000)

Details and structure: see `README_APP.md`.
